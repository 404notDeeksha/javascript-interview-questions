# What is JavaScript?

Javascript is a programming language which is used to convert static web apps into interactive dynamic ones.

## Features

It is lightweight & versatile. It is a dynamically typed language (data type of variable is decided at runtime). It is single threaded. It is weakly typed (variable types are not enforced) & interpreted language. It is Event Driven, asynchronous, & has Rich ecosystem.

## JS Engine

Js engine is a program in web browser which executes js code. In order to enable it, use js file under `<script>` tag in index.html file.
- Chrome - V8
- Firefox - SpiderMonkey
- Safari - JsCore
- Edge - Chakra
